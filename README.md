### 🚗 Quarter-Car Suspension Simulator

A numerical vehicle-dynamics model used to study suspension response  
under different road inputs.

- Sprung and unsprung mass motion
- Suspension displacement
- Wheel response
- Road-bump simulation
- Animated visualisation

`Python` `NumPy` `SciPy` `Matplotlib`
